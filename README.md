# 📚 Book Store Backend API

A complete backend application for an online Book Store built using Node.js, Express.js, and MongoDB. This project provides RESTful APIs for managing books, users, authentication, orders, and other core bookstore functionalities.

The project focuses on backend development, API design, authentication, database operations, and secure request handling. All APIs are tested using Postman.

---

## 🚀 Features

- User Authentication & Authorization
- JWT Token Authentication
- Password Encryption using bcrypt
- CRUD Operations for Books
- User Management
- Order Management
- Protected Routes
- Error Handling
- Input Validation
- RESTful API Architecture
- MongoDB Database Integration
- Environment Variable Configuration
- API Testing with Postman

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (JSON Web Token)
- bcrypt
- dotenv
- Postman
